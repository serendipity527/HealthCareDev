# 快速开始指南

## 🚀 一分钟快速启动

### 方法1：使用启动脚本（推荐）
```bash
# Windows
双击运行 start-chat.bat

# 等待应用启动完成后，浏览器访问：
http://localhost:8080/chat.html
```

### 方法2：使用Maven命令
```bash
# 在项目根目录执行
mvn spring-boot:run

# 启动成功后访问
http://localhost:8080/chat.html
```

## 🔧 技术要点

### WebSocket连接流程
1. 页面加载 → 自动建立WebSocket连接
2. 连接成功 → 订阅 `/topic/messages` 频道
3. 发送消息 → POST到 `/app/chat`
4. 接收回复 → 从 `/topic/messages` 接收

### 消息格式
```json
{
  "sender": "用户",
  "content": "消息内容",
  "timestamp": "2025-11-05T14:30:00"
}
```

### 端口配置
- 默认端口: `8080`
- 修改端口: 编辑 `src/main/resources/application.yml`
```yaml
server:
  port: 8080  # 修改为其他端口
```

## ⚡ 常见问题速查

| 问题 | 解决方案 |
|------|---------|
| 页面显示连接断开 | 确认后端服务已启动 |
| 消息发送无响应 | 查看浏览器控制台(F12)错误信息 |
| 端口被占用 | 修改application.yml中的端口号 |
| Maven构建失败 | 检查JDK版本是否为17或更高 |

## 🎨 界面特色

- **渐变色主题**: 紫色渐变视觉效果
- **气泡对话**: 类似微信/Telegram的对话气泡
- **动画效果**: 消息淡入动画、按钮悬停效果
- **响应式设计**: 适配不同屏幕尺寸
- **状态指示**: 实时显示连接状态
- **自动滚动**: 新消息自动滚动到底部
